# Backlog MVP priorisé (méthode RICE)

Le backlog MVP de DelishAfrica est priorisé selon la méthode **RICE** (Reach, Impact, Confidence, Effort). Le score de chaque fonctionnalité est calculé avec la formule : *(R × I × C) / E*.
